**Sales Dashboard – Power BI Project**

This project is an interactive Sales Dashboard created using Power BI to analyze sales performance and generate useful business insights.
The dashboard helps to understand sales trends, profit, customer behavior, and product performance using dynamic visualizations.

**Project Features**

1) Total Sales, Profit, and Quantity KPI cards

2) Sales analysis by Category and Sub-Category

3) Region / State wise sales performance

4) Monthly and yearly sales trend

4) Top selling products

5) Customer wise sales analysis

6) Interactive filters and slicers

**Tools & Technologies Used**

1) Power BI Desktop

2) Power Query (Data Cleaning & Transformation)

3) DAX Functions

4) Excel / CSV Dataset

5) Data Modeling

**Key Insights**

1) Identified top performing products and categories

2) Found highest sales region

3) Analyzed profit vs sales trend

4) Observed monthly growth pattern

5) Detected low profit products

**Files Included**

1) Sales_Dashboard.pbix

2) Dataset file

3) Dashboard Screenshot

4) README.md

**Objective**

To build an interactive Business Intelligence dashboard that helps in making data-driven decisions using sales data.
